# develOP — Logic Core v3

Portal SaaS multi-tenant + landing page para develOP, agencia
de desarrollo web e IA en Tucumán, Argentina.

## Documentos importantes

- `STATUS.md` — Estado real de funcionalidad
- `AGENTS.md` — Contexto para agentes de IA (Antigravity/Codex)
- `prisma/schema.prisma` — Modelo de datos
- `src/lib/data/premium-modules.ts` — Catálogo source of truth de módulos premium

## Stack

- Next.js 16 (App Router)
- TypeScript estricto
- Prisma + Neon PostgreSQL (sa-east-1)
- NextAuth v5
- Tailwind CSS 4
- Framer Motion
- Three.js + R3F
- Anthropic SDK (Claude Haiku 4.5)
